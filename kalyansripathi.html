Social Faker

socialfaker.com

Social Faker helps detect fake users on Facebook®. Impersonating others profiles is a big problem on Facebook®. Social Faker allows you to detect users with the same name and report them if they are impersonating your profile. It also sends the users weekly updates on any new fake users detected.

Superstar

Superstar is a voting website I built for Aquent. It uses Google's authentication to verify users identity and gives the users an ability nominate people and recognise them for their contribution. Also users can vote for the nominees which determines the winning nomination for the quarter that the company recognizes as the superstar for the quarter.

Battlescripts

battlescripts.net

Battlescripts was a website built with colleagues at work. It is a javascript gaming engine. Users can pick from a list of games to play and write code to create players to compete against other players. Users can register using any social network and can save players they created and play against other registered players.AngularJS, REST, PHP, hello social API and other technologies were used to create this website.

FS Dashboard

FS Dashboard

FS Dashboard is an ongoing project. It is an android app that starts on device boot up and goes to a pre-configured url and relays its MAC ID. The dash board website can be used to configure different widgets for different devices based on the MAC ID relayed by the app. The idea is to be able to use an android device like a dashboard say a calendar, to show weather, to monitor Github commits or any other available widgets and require no configuration on the device at all.

The user will be able to login to the dashboard control center website and customize all his available devices and what widgets should be shown on them.
